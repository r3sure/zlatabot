## Технический стек

| Компонент | Технология |
|-----------|-----------|
| Бот | aiogram 3.x |
| AI | Groq API (llama-3.3-70b-versatile, бесплатно) |
| Астрология | kerykeion (обёртка над Swiss Ephemeris) |
| БД | SQLite 3 |
| SVG→PNG | Playwright Chromium |
| Изображения луны | Pillow (PIL) |
| Планировщик | APScheduler |
| Прокси | HTTP :10809 (Telegram заблокирован) |
| AI-тексты | ~30k токенов/мес на пользователя |

**Проверки:** `has_premium_access()` = trial или full. `is_premium()` = только full.
